# 快速开始

paste 是一个极简的文本中转站:访问 `/任意名字` 打开一个全屏文本框,内容变动时自动保存到本地 txt 文件,方便在电脑和手机之间互相复制内容。

它零依赖、只用 Python 标准库,整个服务就是单文件 `app.py`。

## 运行

克隆仓库后直接运行:

```bash
python3 app.py
# 默认监听 0.0.0.0:8000,数据存到 ./notes/
```

启动后终端会打印访问地址和数据目录。用浏览器打开:

- `http://localhost:8000/` — 根路径 `/` 会重定向到 `/note`
- `http://localhost:8000/任意名字` — 打开(或新建)一份对应的 paste

在文本框里输入内容,停顿一下就会静默保存。换台设备打开同一个 URL,就能看到刚才的内容。

## 环境变量

三个可选环境变量控制监听地址和数据目录:

| 变量        | 默认       | 说明             |
| ----------- | ---------- | ---------------- |
| `HOST`      | `0.0.0.0`  | 监听地址         |
| `PORT`      | `8000`     | 监听端口         |
| `NOTES_DIR` | `./notes`  | txt 文件存放目录 |

示例:

```bash
PORT=9000 python3 app.py            # 自定义端口
NOTES_DIR=/data python3 app.py      # 把数据存到 /data
```

## 数据存哪

每份 paste 对应 `NOTES_DIR` 下的一个 `.txt` 文件,例如 `/note` 对应 `notes/note.txt`。这些是你的实际内容,不会进版本库(仓库 `.gitignore` 已忽略 `notes/`)。

想了解路径映射、自动保存和安全设计的细节,见 [工作原理](/guide/how-it-works)。要部署到服务器,见 [部署](/deploy)。
